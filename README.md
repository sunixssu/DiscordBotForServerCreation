# Discord Bot for Channels Creation

Это Discord бот для создания каналов внутри Discord сервера по требованиям пользователя. Он работает за счет Discord API и Mistral AI REST API

У бота есть всего одна команда: `/setup-channel`. После ее вызова появится всплывающее окно, где можно указать требования для каналов: количество текстовых и голосовых каналов, а также их предназначения. По этим предназначениям ИИ подберет наиболее подходящие названия и применит их к созданным каналам.

После указывания необходимого количества текстовых и голосовых каналов, а также описания предназначения этих каналов, будут бот отправит сообщение, что он приступил к работе и через несколько секунд будут созданы каналы, которые будут разделены на категории: **Text** и **Voice**. В названиях каналов, как приятный бонус, будут эмодзи, чтобы сервер выглядел красивее

Бот работает за счет библиотеки [discordgo](https://github.com/bwmarrin/discordgo)

# Установка

Для установки необходимо в терминале прописать

```
go get sunixssu/DiscordBotForServerCreation
```

# Использование

Для начала нужно будет создать бота на [сайте Discord](https://discord.com/developers/applications).

Чтобы использовать бота, необходимо создать файл `data.env`, в котором нужно указать **CLIENT_ID**, **PUBLIC_KEY**, **BOT_TOKEN**, **AI_API_KEY**.

- CLIENT_ID - это ID сервера, на котором находится бот. Его можно скачать нажав ПКМ по серверу и нажать "Копировать ID". Для этого должен быть включен режим разработчика, который включается в настройках
- PUBLIC_KEY - это ключ, который можно получить, создав бота на [этом сайте](https://discord.com/developers/applications). Там будет отдельный пункт "Публичный ключ", который здесь и нужен будет
- BOT_TOKEN - это токен бота, который можно получить, создав бота на [этом сайте](https://discord.com/developers/applications). В разделе "Бот" его можно будет найти и скопировать
- AI_API_KEY - это api-ключ для ИИ, который будет генерировать наиболее подходящее название каналов. Я использовал mistral.ai

После создания бота и добавления его на сервер, запускаем программу

```
go run main.go
```

Если в терминале увидели команду **Bot is ready!**, то все работает, и ботом можно пользоваться!
